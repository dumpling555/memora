数据流详解

1. 照片列表获取

  • 前端 GET /api/photos?folder=xxx&q=yyy
  • Flask 查询 image_analysis 表
  • 按 folder 字段 INSTR 匹配
  • 按 q 在 6 个字段中 LIKE 全文搜索
  • 返回 JSON 数组 (id, name, tags, ...)

2. 文件夹列表获取

  • 前端 GET /api/folders
  • 从 file_path 提取第一级子目录
  • DISTINCT + COUNT 聚合
  • 返回 [{name, count}, ...]

3. 图片文件服务

  • 前端 GET /api/file/<id>
  • 查 DB 获取 file_path
  • 剥离 NAS 前缀，拼接本地路径
  • send_file() 流式返回

核心功能

🔍 文件夹筛选

从数据库动态提取文件夹列表，支持多选切换，默认展示 printscreen 文件夹

📝 全文搜索

在 overview / extracted_text / tags / file_name / file_path 等 6 个字段中 LIKE 模糊匹配

🕓 图片预览

点击缩略图打开模态弹窗，展示原图 + AI 识别描述 + 标签

🕑 时间线视图

按日期分组展示照片，右侧边栏固定定位，快速定位历史照片

🌄 响应式网格

2~5 列自适应布局 (CSS Grid)，hover 动画卡片效果

📊 零依赖部署

Flask + sqlite3 + HTML，无需额外数据库或构建工具，python app.py 即可运行
