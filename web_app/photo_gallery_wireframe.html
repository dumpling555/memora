4. API 接口
方法端点说明参数
GET/渲染主页 (index.html)—
GET/api/photos获取照片列表，支持多文件夹筛选与全文搜索q (搜索词), folder (文件夹名, 可多选)
GET/api/folders从数据库动态提取所有文件夹名称及图片数量—
GET/api/file/<id>根据照片ID返回原始图片文件photo_id (路径参数)
5. 数据库表结构 (image_analysis)
字段名说明
id主键 (INTEGER PRIMARY KEY)
file_name文件名 (如 screenshot_001.png)
file_path完整NAS路径 (如 \\myDiskstation\photo\电脑\printscreen\...)
overviewAI生成的图片内容概述
extracted_textOCR提取的文本内容
other_info其他AI分析信息
tags标签 (逗号分隔, 如 风景,夕阳,天空)
width图片宽度 (px)
height图片高度 (px)
format图片格式 (如 PNG, JPEG)
created_at创建时间 (TIMESTAMP)
